# chatRoom
图形界面聊天室
## 1.简介
仿照QQ聊天工具，基于Java语言开发的多线程图形界面聊天室。
## 2.环境
IDEA下编写，Maven工具管理
## 3.功能

 - 实现持久通信，通过“退出”按钮退出登陆
 - 实现指定群发和私发功能
 - 实现服务端使用多线程处理客户端信息
 - 实现客户端读写同时进行
